# Smoke Patrol 烟量巡逻官 - 项目记忆

## 项目概述
- **名称**: Smoke Patrol 烟量巡逻官
- **目的**: 帮助用户控制烟量的前端小工具
- **风格**: 美式卡通漫画游戏风（Bangers 字体、粗描边、半调网点背景、对话气泡）
- **技术**: 单 HTML 文件，零依赖，localStorage 持久化，Python http.server 本地服务
- **端口**: 8089

## 核心逻辑
1. 用户设置"每 X 分钟可抽 Y 支烟"
2. 点击"现在能抽烟吗？"按钮查询
3. 系统基于累积配额判定：每过一个间隔累计 Y 支，可跨多个间隔叠加
4. 每次确认可抽时自动扣减 1 支并计入当日统计，同时显示当前最多可抽总量
5. 跨天自动重置当日计数

## 数据结构 (localStorage)
| 字段 | 说明 |
|---|---|
| intervalMinutes | 间隔分钟数 |
| allowedCount | 每次可抽支数 |
| tokenCount | 当前累计可抽配额 |
| lastAccumulateTime | 上次累计配额的时间戳 |
| todaySmoked | 今日已抽数量 |
| todayDate | 今日日期字符串 |

## 文件结构
